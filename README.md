# Data Communication (24CS430)

Study materials for the **Data Communication** course offered by the Department of Computer Science and Engineering.

## Syllabus Overview

### Unit 1 — Introduction (7 hrs)
- Data Communications, Networks, and Administration
- Network Types
- Network Models: Protocol Layering, TCP/IP Protocol Suite, The OSI Model
- Standards

### Unit 2 — Physical Layer (8 hrs)
- Data and Signals, Digital Signals, Transmission Impairment, Data Rate Limits, Performance
- Digital Transmission: Digital-to-Digital Conversion (Line coding: Polar, Bipolar, Manchester), Analog-to-Digital Conversion (PCM), Transmission Modes

### Unit 3 — Bandwidth & Switching (8 hrs)
- Bandwidth Utilization: Multiplexing
- Transmission Media: Guided and Unguided Media
- Switching: Circuit Switched Networks, Packet Switching, Structure of a Switch

### Unit 4 — Error Detection & Network Layer (8 hrs)
- Error Detection and Correction: Block Coding, Cyclic Redundancy Checksum (CRC), Hamming Distance, XOR
- Network Layer: Performance, IPv4 Addresses

### Unit 5 — Data Link Layer (8 hrs)
- Introduction to Data-Link Layer: Link-Layer Addressing, DLC Services, Data Link Layer Protocols
- Point-to-Point Protocol (Framing, Transition Phases)
- Media Access Control: Random Access, Controlled Access

## Course Outcomes

| CO | Description |
|---|---|
| CO1 | Understand the importance of physical and data link layers of the OSI model |
| CO2 | Analyze and evaluate network topologies and protocols |
| CO3 | Implement different communication protocols |
| CO4 | Explore fundamental issues driving network design |
